# V2 UI Docs

This folder is the canonical location for v2 UI documentation in the clean Nornickel hackathon workspace.

Use this folder for UI-specific specs, implementation plans, notes, and candidate backlogs for both browser UIs:

- ore pipeline UI: `apps/ore_pipeline_web.py`
- talc mask review UI: `apps/talc_review_web.py`

Root-level `docs/plans/`, `docs/specs/`, and `docs/notes/` are reserved for non-UI official task plans, research notes, model cards, benchmarks, and handoff material. Operational launch/test snippets can remain in top-level files such as `COMMANDS.md` and `SMOKE_TESTS.md`, but detailed UI behavior belongs here.
