# ChangeLog

## 2026-07-03

- Added thumbnails to the ore pipeline History page; each table row now shows a small run preview, and clicking it opens a modal with the larger stored preview image.
- Fixed the ore pipeline Edit & Recalculate dialog so it refreshes the full run payload before loading editor masks/images, re-registering artifact URLs after server restarts or stale browser state, and shows a localized editor error instead of a blank canvas when prerequisites are missing.
- Moved talc review `Save` and `Save & Next` actions from the right details panel into a top-right action group beside the canvas toolbar.
- Added `Sulfide mask (sulfide/non-sulfide mask segmentation)` to the talc review app's `Base image` selector so reviewers can inspect the sulfide mask as the canvas background while editing talc.
- Added cancelable ore pipeline runs: pressing `Start` swaps the control to a red `Stop` button, `POST /api/runs/{run_id}/cancel` marks active jobs as canceling/canceled, ML subprocess runs are terminated cooperatively, and the UI polling/history flow treats canceled runs as terminal.
- Added a `show tiling` viewer checkbox to the ore pipeline UI. Preprocessing now records a tile manifest for scaled/large images, runs carry it forward, and the canvas can overlay the tile grid on original/preprocessed/result views.
- Enabled all ore pipeline preprocessing options by default and persisted the user's preprocessing preset in local storage, including after page reloads and upload clears.
- Added upload/preview-preparation progress to the ore pipeline UI: large image uploads now show byte-transfer progress, then a localized preparation phase while the server decodes and scales preview pyramids; manual preprocessing preview updates use the same visible progress row.
- Added inline upload warnings to the ore pipeline drop zone: unsupported dropped/selected file extensions are rejected client-side with localized PNG/JPEG/TIFF/RAW guidance, and backend upload failures use the same warning area.
- Added SAM2 point hover previews to the talc browser review app: in `SAM2 point` mode, holding the cursor still for 2 seconds runs SAM2 into an orange preview overlay, and `Apply SAM2` applies that preview or runs/applies the current hover point.
- Changed the ore pipeline History page to a table with filename, date, ore classification, sulfide/non-sulfide and class fractions, plus `Load` and `Remove` actions; `Remove` deletes completed/failed run artifacts through `DELETE /api/runs/{run_id}` while leaving uploads intact.
- Added Delete-key shape removal to the talc browser review app: selected completed polygons or rectangles can be removed with Delete/Backspace, while text fields keep normal delete behavior.
- Added slug URLs to the ore pipeline UI: `/` redirects to `/workspace`, `/workspace` and `/history` are direct-loadable pages, tab clicks update the URL, and browser back/forward restores the matching page.
